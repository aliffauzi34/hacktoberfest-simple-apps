# 🔗 PHP Slug Generator (Bootstrap)

Convert any title or sentence into a clean **URL slug**.  
Options: lowercase, custom delimiter, stopword removal (EN/ID), and max length.

## 🚀 Run
```bash
php -S localhost:8080

Custom delimiter (default -)
